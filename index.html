How to use?

1. Start by understanding the rules for solving the Sudoku puzzle.

2. Fill the values in given sudoku puzzle. Keep in mind that, there can never be any same value in a single row or a column. Even it is not possible in a 3*3 matrix (outlined in a dark).

3. Click on the "Solve" button. All the values get automatically filled in the puzzle that will be your final answer.

4. You can erase all the values by just clicking on the "Reset" button.

5. Follow the above steps again to solve more puzzles.
